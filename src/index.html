Project Highlights

MBTA Near Me

Boston, Massachusetts

A fully functional progressive web app that provides predictions of the MBTA schedules based on location. It provides the user with an interactive map that retrieves up to the second schedules of all the available public transport options availabe near the user. This is a react app built using the MBTA's V3 API, MapLibre and a MongoDB back end.

OECD TRACE

Helsinki, Finland

Finland was the FIRST country in the EU to have implemented the TRACE OECD program which enables countries to track international dividend distribution from the source through intermediaries to the final dividend recipient. This implementation has set the standard for the remaining European states to follow. The program has enabled the Finnish Tax Administration uncover vast tax evasion schemes happening on a global scale with an estimated €80 million in uncollected taxes.

KIVE

Helsinki, Finland

Multiple legacy systems were sunset and replaced with the KIVE project which handles all of Finland's real estate taxation needs for it's ~4 millions parcels of land and ~3 million buildings. This project was part of a bigger award winning modernization effort for the Finnish Tax Administration where it was named the best project in Finland for 2020 and placed 2nd place internationally for Mega Projects (budget > €200 million).

Georgia DRIVES

Atlanta, GA, USA

The Driver Record and Integrated Vehicle Enterprise System (DRIVES) was implemented as a new system for the Georgia Department of Motor Vehicles to replace their legacy system. This effort enabled the agency to conduct its daily business transactions in a much more efficient manner and reduced wait times for residents. The system is one of the largest vehicle registration and titling systems managing an astonishing 35 million vehicles and 16 million owners. The system was recognized as one of the Top Technology projects at the 2020 Georgia Tech Show and received national awards at the American Association of Motor Vehicle Administrators (AAMVA) in 2020
